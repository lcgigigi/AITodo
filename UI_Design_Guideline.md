# 现代教育/效率看板 UI 设计规范 (Design Guideline)

基于 "Online Learning Education Dashboard" 参考图的设计语言解析。此文档作为后续前端 UI 组件开发与优化的基准参考。

## 1. 核心设计理念 (Core Concept)
- **年轻、活力、现代**：通过高明度、低饱和的辅助色，结合高饱和的主题色，打破传统效率软件的沉闷感。
- **卡片化与呼吸感 (Card-based & Airy)**：使用大圆角和弥散阴影的白色卡片，放置在极浅灰色的底层背景上，模块之间留出充足的间距。
- **视觉趣味性 (Playfulness)**：通过角落的半透明几何/圆形色块点缀（Blobs）和抽象几何图形，增加界面的灵动感，避免纯数据的枯燥。

## 2. 色彩系统 (Color Palette)
- **底层背景色**：极浅冷灰/灰蓝 (`#F8F9FB` / `#F3F4F6`)，使纯白卡片能自然凸显。
- **卡片背景**：纯白 (`#FFFFFF`)，作为信息承载的主体容器。
- **主标题与核心数字**：深藏青色/黑灰 (`#0F172A` / `#111827`)，提供极其清晰的视觉焦点和对比度。
- **四种核心点缀色 (Accent Colors)**：
  - 🟠 **活力橙/陶土橘 (Orange)**：如 `#D97736` 或 `#E07A5F`，用于提醒、新动态展示（如大数字 19）。
  - 🟣 **优雅紫/薰衣草 (Purple)**：如 `#7C3AED` 或 `#6B46C1`，用于主要行动按钮（CTA）、进度条、重点功能入口。
  - 🟢 **清新绿/薄荷绿 (Mint Green)**：如 `#10B981`，用于成功状态、总计数据、积极的反馈。
  - 🔵 **商务蓝/清爽蓝 (Blue)**：如 `#2563EB`，用于常规链接、标签或次要操作。
- **浅色底板 (Pastel Backgrounds)**：主颜色的极低透明度版本（如浅紫 `#F5F3FF`，浅橙 `#FFF7ED`，浅绿 `#ECFDF5`），广泛用于卡片内装饰圆块、标签底色和部分模块底色。

## 3. 造型与组件风格 (Shapes & Components)
- **圆角 (Border Radius)**：
  - 容器级大卡片：`20px - 24px`
  - 内部小卡片/列表项：`12px - 16px`
  - 按钮/标签：通常使用 `8px - 12px` 或 `999px` (全圆角药丸状)。
- **阴影 (Shadows)**：
  - 弱化或弃用生硬的实线边框，改用极其柔和的下沉弥散阴影：`0 4px 24px rgba(0, 0, 0, 0.04)`，制造悬浮的呼吸感。
- **列表项 (List Items)**：
  - **默认状态**：带有极其轻微的灰色/浅蓝色背景（如 `#F8FAFC`），不使用边框。
  - **交互状态**：悬浮 (Hover) 时背景微微加深（如 `#F1F5F9`）或向上轻微浮动（`translateY(-1px)`）。
- **图标与头像 (Icons & Avatars)**：
  - 辅助图标常放置在带有柔和背景色的圆角矩形容器中（Icon Wrapper）。
  - 头像组经常带有纯白色外圈描边（Stroke），形成层叠效果。

## 4. 排版体系 (Typography)
- **字重对比极强**：大号、加粗（`Weight: 800+`）的无衬线字体用于核心数字和标题。
- **辅助文字淡化**：描述性文字、副标题使用浅灰色（如 `#94A3B8` 或 `#64748B`），字号偏小（`12px - 13px`），形成强烈的主次对比，方便用户快速扫视（Scanning）。

## 5. 装饰元素特征 (Decorative Elements)
- **Blobs (角落圆形色块)**：在白色数据卡片的对角线（如左上、右下）放置部分露出（使用 `overflow: hidden` 切割）的浅色透明圆块。这是该风格最显著的视觉符号。
- **几何图形插画**：通过简单的基础几何体（方形、菱形、圆柱）拼贴成的 2D/2.5D 插图替代写实照片。
